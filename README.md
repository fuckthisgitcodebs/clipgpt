# EdgeClipPlus

Samsung Edge Clipboard History clone + expanded features.

## Features (Current)
- Foreground clipboard monitoring service
- Room DB clipboard history storage
- Overlay edge swipe trigger
- Overlay panel activity (UI prototype)
- Accessibility service scaffold

## Build
Push to GitHub -> GitHub Actions builds APK artifact.

## Install
Download artifact `app-debug.apk` from Actions -> Artifacts.

## Permissions Needed
- Overlay permission
- Accessibility service enable
- Foreground service notification permission (implicit)

## Roadmap
- True slide-out animated panel (Compose animation)
- Copy/paste automation via accessibility actions
- Pin/favorite UI
- Search UI
- Multi-device sync
